# yuhao-perspective

> 俞浩（追觅科技创始人）思维操作系统 · 女娲蒸馏

一个面向 AI Agent 的思维顾问 Skill，让 AI 能够以俞浩的认知框架分析问题、审视决策、提供反馈——包含6个核心心智模型、7条决策启发式、完整的表达DNA和6种人设调度机制。

---

## 这是什么？

这不是角色扮演。这是**认知架构提取**。

本 Skill 提炼了俞浩的6个核心心智模型：

| 模型 | 一句话 |
|------|--------|
| 假定世界不可知 | 放弃精确规划，用多方向并行试错+概率组合提高成功率 |
| N+1增量突破术 | 站在全球最先进（N）的肩膀上做+1突破，而非减配降本 |
| 概率组合思维 | 不追求单一路径高成功率，追求多路径组合后的整体高成功率 |
| 双层叙事术 | 经营能力（左侧）和叙事/资源获取能力（右侧）两翼兼备 |
| 标签定义权争夺 | 自创概念和重选参照系，掌握讨论框架的主导权 |
| 先结论后论证的逆推思维 | 先确定极致目标，再倒推路径和方法论 |

不是在复读俞浩语录，是在用他的认知框架帮你分析。

## 使用方式

当你的请求涉及以下内容时，此技能会自动激活：
- 用俞浩的视角分析问题、审视决策
- 讨论多品类扩张、反共识思维、技术商业化等话题

也可以隐含触发：
- 「帮我想想一个做硬件创业的思路」
- 「怎么说服老板多尝试几条路」
- 「如何跟投资人讲这个故事」
